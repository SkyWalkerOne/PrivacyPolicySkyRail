SkyRail: Политика конфиденциальности / SkyRail: Privacy Policy

1. Аккаунт и данные игрока / Account and Player Data

Собираемые данные / Collected Data:
Приложение SkyRail использует минимально необходимую информацию для работы аккаунтов и предоставления игровых функций. Мы собираем и обрабатываем:

  • Имя пользователя (username) — уникальный идентификатор игрока в игре, создаваемый пользователем.
  • Идентификаторы устройства (например, Advertising ID) — используются для предотвращения мошенничества, безопасности, и корректной работы рекламы.
  • Игровые сообщения в мультиплеере — временно обрабатываются для работы игрового чата.
Все данные передаются через защищённое шифрованное соединение.

SkyRail only collects the minimal data required for account functionality:
  • Username — the player's unique in-game identifier.
  • Device identifiers (e.g., Advertising ID) — used for fraud prevention, security and proper ad functioning.
  • In-game multiplayer chat messages — processed temporarily to enable multiplayer communication.
All data is transferred through secure encrypted connections.

Создание аккаунта / Account Creation:
Для создания аккаунта требуется только имя пользователя. Электронная почта и номер телефона не используются и не запрашиваются.
Only a username is required to create an account. No email or phone number is used.

Удаление аккаунта / Account Deletion:
Пользователь может запросить удаление аккаунта через страницу: https://skywalkerone.github.io/SkyRailAccountDeletionInfo/.
For account deletion requests, the user may contact support via the link above.

Для удаления аккаунта пользователь должен предоставить свое имя пользователя и доказательства владения аккаунтом (например, скриншоты или видеозапись игрового процесса).
